
Permissions/Privileges:

| Level                         | Description                                     | Examples                                    |
| ----------------------------- | ----------------------------------------------- | ------------------------------------------- |
| **Database-Level Privileges** | Control what users can do *within a database*   | Connect, create schemas, temporary tables   |
| **Object-Level Privileges**   | Control what users can do *on specific objects* | Select from tables, execute functions, etc. |

object-level access - seeing schema or metadata
data-level access   - querying rows
